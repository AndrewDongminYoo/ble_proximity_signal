## Overview

This package provides a **foreground BLE scanner** that maps RSSI to **strength values (0..1)** for up to 5 target identifiers (e.g., device ID/UUID).

## Key Features

- Supports foreground BLE scanning
- Monitors up to 5 target identifiers
- Normalizes RSSI to intensity values in the 0..1 range for display

## Integration tests üß™

Very Good Flutter Plugin uses [fluttium][fluttium_link] for integration tests. Those tests are located
